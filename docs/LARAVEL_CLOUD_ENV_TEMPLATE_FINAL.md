# LARAVEL_CLOUD_ENV_TEMPLATE_FINAL

Target: `C:\deploy\adminpanel`

Gunakan template ini sebagai acuan ENV production di Laravel Cloud. Jangan isi secret langsung di repository.

## APP

```env
APP_NAME="eMart"
APP_ENV=production
APP_KEY=
APP_DEBUG=false
APP_URL=
APP_TIMEZONE=UTC
APP_LOCALE=en
APP_FALLBACK_LOCALE=en
APP_FAKER_LOCALE=en_US
```

## Catatan penting

- Isi kedua varian ENV Firebase yang mismatch agar source lama tetap jalan:
  - `FIREBASE_APIKEY` dan `FIREBASE_API_KEY`
  - `FIREBASE_MESSAAGING_SENDER_ID` dan `FIREBASE_MESSAGING_SENDER_ID`
- File `storage/app/firebase/credentials.json` tidak boleh di-commit.
- Materialisasi credential dilakukan saat release/deploy, bukan disimpan permanen di repo.
